## 2. What Requires Technical Knowledge

### Mandatory Technical Requirements

| Feature | Requirement | Difficulty | User Type |
|---------|-------------|------------|-----------|
| Run JARVIS at all | Python 3.11+, pip, uvicorn | Hard | Developer only |
| OpenAI integration | OPENAI_API_KEY in .env | Medium | Developer |
| Outlook email | Azure AD app registration, client_id, tenant_id, client_secret, redirect URI | Very Hard | Enterprise dev |
| IBKR LIVE portfolio | IB Gateway installed locally, running on port 4001, IBKR_PORT in .env | Hard | Trader/Developer |
| IBKR remote bridge | secure_bridge.py running locally, ngrok tunnel, IBKR_BRIDGE_URL + IBKR_BRIDGE_TOKEN on Railway | Very Hard | Developer/Infra |
| Voice (TTS) | ELEVENLABS_API_KEY in .env | Medium | Developer |
| WhatsApp | TWILIO_ACCOUNT_SID + AUTH_TOKEN + WHATSAPP_NUMBER in .env | Medium | Developer |
| Google Calendar | GOOGLE_CLIENT_ID + CLIENT_SECRET, OAuth consent screen | Hard | Developer |
| Deploy to cloud | Railway account, Dockerfile, env var management | Medium | Developer/DevOps |
| Enable fitness tabs | Find Modules gear icon in Overview | Low | Technical user |
| Memory backfill | Know what "backfill" means, understand graph memory concept | Medium | Technical user |

### Required Terminal Commands (No UI equivalent)
```bash
# Start JARVIS
uvicorn main:app --host 0.0.0.0 --port 8000

# Start local bridge (separate terminal)
python -m opsx.bridge.secure_bridge

# Set up ngrok tunnel (separate terminal)
ngrok http 7755

# Install dependencies
pip install -r requirements.txt
```

### Required Environment Variables (Non-obvious)
```
# These are not explained in the UI at all:
ENABLE_REMOTE_IBKR_BRIDGE=true        # Only documented in code comments
IBKR_BRIDGE_URL=https://xxx.ngrok.io  # Generated dynamically by ngrok
IBKR_BRIDGE_TOKEN=xxx                 # Generated by secure_bridge.py on startup
IBKR_PORT=4001                        # Not 7497 (paper) or 7496 (banned)
OPENAI_MODEL=gpt-4o-mini              # Not obvious which models are supported
ENCRYPT_PORTFOLIO_CACHE=true          # Not clear if this actually does anything
```

## 3. Hidden Setup Requirements

### Outlook / Microsoft Graph
1. Create Azure AD application in Microsoft Entra
2. Set application type to "Public client" or "Web"
3. Add redirect URI: `https://your-app.railway.app/auth/microsoft/callback`
4. Copy Application (client) ID → `OUTLOOK_CLIENT_ID`
5. Copy Directory (tenant) ID → `OUTLOOK_TENANT_ID`
6. Create client secret → `OUTLOOK_CLIENT_SECRET`
7. Add Graph API permissions: `Mail.ReadWrite`, `Mail.Send`, `Calendars.ReadWrite`
8. Grant admin consent

**None of this is documented in the UI. Zero in-app guidance.**

### IBKR Bridge
1. Install Interactive Brokers Trader Workstation or IB Gateway
2. Enable API access in TWS/Gateway settings
3. Set port to 4001 (live) or 4002 (paper)
4. Set `Read-Only API = checked`
5. Start secure_bridge.py locally
6. Note the bridge token printed on startup
7. Install and start ngrok on port 7755
8. Copy ngrok URL → `IBKR_BRIDGE_URL` on Railway
9. Copy bridge token → `IBKR_BRIDGE_TOKEN` on Railway
10. Set `ENABLE_REMOTE_IBKR_BRIDGE=true` on Railway

**10 manual steps. Zero in-app guidance.**

### Google Calendar
1. Create Google Cloud project
2. Enable Google Calendar API
3. Create OAuth 2.0 credentials
4. Set authorized redirect URI
5. Copy client ID and secret to .env

**5 manual steps. Zero in-app guidance.**

## 5. How to Make It Plug-and-Play

### Tier 1: Immediate (0 external dependencies)
These features should work the moment a user opens the dashboard. They currently do, but need clearer empty states.

**Fixes needed:**
- Show "Start here: Add your first task" in Productivity when empty
- Show "Chat with JARVIS to get started" in Overview when no data
- Show "Add a golf round to see insights" in Golf when no rounds
- Show "Add items to track" in Life when empty

### Tier 2: One-step connection (requires only API key)
These should be set up via an in-app Settings panel:
- OpenAI key → Chat, Brief, Analysis
- ElevenLabs key → Voice TTS
- Twilio → WhatsApp notifications

**What's needed:**
- Settings tab with API key input fields
- Connection test button ("Test connection")
- Status indicator showing connected/disconnected

### Tier 3: OAuth connection (requires account, guided flow)
- Google Calendar → should show "Connect Google Calendar" card with step-by-step in-app guide
- Microsoft/Outlook → should show "Connect Outlook" card with step-by-step in-app guide

**What's needed:**
- Setup wizard: click "Connect" → redirect → callback → success/error confirmation
- Currently this partially works but has zero in-app explanation before the redirect

### Tier 4: Infrastructure connection (requires local software)
These can never be fully plug-and-play but need a proper setup guide:
- IBKR: "Here's how to connect your IBKR portfolio (5 steps)"
- Should open a modal with: step 1 → step 2 → step 3 → verify connection
